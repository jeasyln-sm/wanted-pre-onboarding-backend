# 📑wanted-pre-onboarding-backend

### ✒️주요 기능 
* 채용 공고 등록
  * 회사는 아래 데이터와 같이 채용공고를 등록합니다.
  * 회사_id는 DB에 미리 등록된 데이터를 가져와 선택합니다.
  * 그 외 채용 포지션, 채용 보상금, 채용 내용, 사용 기술은 직접 입력합니다.
